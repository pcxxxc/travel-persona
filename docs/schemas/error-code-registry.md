# 错误码分配表

旅格统一错误码体系。格式: `TP-NNNN`。

## 核心原则

用户端不得展示"降级模式""fallback""Agent超时""智能体失败"等工程状态（总纲1.5）。内部错误不应对用户可见。

## VALIDATION (TP-1xxx)

| 错误码 | 含义 | 用户可见 | 用户消息 |
|--------|------|---------|---------|
| TP-1001 | PersonaScore 缺少必填维度 | 否 | - |
| TP-1002 | 维度值超出 [0,1] 范围 | 否 | - |
| TP-1003 | 问卷答案值不在允许范围内 | 是 | "请选择一个有效的选项" |
| TP-1004 | 预算格式或范围错误 | 是 | "预算金额格式不正确" |
| TP-1005 | 天数超出允许范围 | 是 | "旅行天数请填写 1-60 天" |
| TP-1006 | 必填字段缺失 | 是 | "请填写必要信息" |
